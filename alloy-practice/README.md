## 検証対象のOSS
npmパッケージであるpicoaudioを検証対象とした。

picoaudioは、web上でSMFを再生することができるパッケージで、SMfのパースや8bit音での楽曲の再生を行うことができる。

このOSSのリポジトリは https://github.com/cagpie/PicoAudio.js/tree/master にあり、ドキュメントはpicoaudioリポジトリ内のREADME.mdにある。
